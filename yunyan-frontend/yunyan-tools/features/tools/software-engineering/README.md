# Software Engineering Diagram Tool

## Purpose
- Generate software-engineering style diagrams from AI/manual input.
- Support local-first editing with backend-ready runtime contract.

## Module Structure
- `components/workspace`: workspace composition.
- `constants`: preset metadata.
- `services`: API and runtime contract.
- `types`: tool types.

## Runtime
- Runtime contract export: `softwareEngineeringRuntimeContract`.
